Smart Travel Itinerary Generator

GitHub   Live Demo   Figma Design

Overview

Planning trips can be a hassle—jumping between apps just to figure out where to go, where to stay, and what to do. I built this tool to simplify all that by creating personalized itineraries using real-time travel data. Ideal for solo travelers and spontaneous planners who want smart suggestions without the headache.

Solution & Features

  • Automated Itinerary Creation: Automatically gathers and organizes flights, stays, and plans based on user input.
  • Smart Recommendations: Recommends personalized activities using travel history and preferences.
  • Live Data Integration: Pulls real-time flight and hotel info via travel APIs.

Tools & Tech

Python & LangChain • OpenAI GPT-3 • Travel APIs
